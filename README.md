# ふたりのひだまり 🏡

ふたりで育てる、ふたりだけのペット。

2人専用の恋人向けWebアプリです。共有ペット「もこ」(名前は変更可)を中心に、
ミニゲーム・質問トーク・実績・ごほうび券を通じて、2人の毎日がペットの成長に変わります。

## できること

| 機能 | 内容 |
|---|---|
| 🐣 ペット育成 | レベル・5段階進化・満腹度・きげん・性格(4軸)・セリフ・きせかえ |
| 💭 いしんでんしんワード | 同じお題に同じ言葉を答えられるか(協力・5ラウンド) |
| 🪙 裏表心理コイン | 相手の心を読む交互対戦(5ラウンド) |
| 🃏 しんけいすいじゃく | ノーマル50枚/ハード100枚。ペア数で勝負 |
| 🔢 かずあてゲーム | 1〜100のかくし数字を交互に予想(各3回) |
| 🌈 UNO | 108枚フルデッキの2人対戦。UNO宣言演出つき |
| 💬 質問トークデッキ | 毎日の1問+36問。2人とも答えると見せ合える |
| 💗 ハートのかけら | 2人が同じ日に遊んだ時だけ +1。限定きせかえと交換 |
| 🏅 実績・称号 | 24実績・8称号。遊び方に応じて自動解除 |
| 🎟️ ごほうび券 | 「マッサージ券」など現実のごほうびと交換できる券 |
| 📖 きろく | 全履歴・ペットの成長年表 |

## 技術構成

- **ホスティング:** GitHub Pages(無料・静的)
- **バックエンド:** Firebase Authentication + Realtime Database(無料枠)
- **フロント:** 素のHTML/CSS/JavaScript(ES Modules)。**ビルド不要**
- iPhoneのSafari+GitHubモバイルWeb編集だけで開発・運用できる構成

## ディレクトリ

```
futari-hidamari/
├ index.html            エントリ
├ manifest.json         PWA設定(ホーム画面追加用)
├ database.rules.json   Firebaseに貼るセキュリティルール
├ assets/               アイコン
├ css/style.css         全スタイル(昼/夜テーマ)
└ js/
   ├ config.js          ★あなたが書き換える唯一のファイル
   ├ main.js            起動・購読・日次処理
   ├ core/              基盤(認証/DB/UI/経済/実績/対戦/履歴/プレゼンス)
   ├ pet/               ペットエンジン(ロジック/セリフ/SVG描画)
   ├ games/             ゲームプラグイン(1ファイル=1ゲーム)
   ├ screens/           5タブの画面
   └ data/              マスターデータ(お題/質問/実績/セリフ/きせかえ)
```
