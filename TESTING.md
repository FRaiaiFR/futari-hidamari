# TESTING.md — 動作確認手順とテスト項目

2台のスマホ(または スマホ+PCブラウザのシークレットウィンドウ)で、
それぞれ別のアカウントでログインして確認します。

## 1. 起動・ログイン

- [ ] URLを開くと「ひだまりを あたためています…」→ ログイン画面が出る
- [ ] キャラカードを選ぶ→パスワード入力→「はいる」でホームに入れる
- [ ] 間違ったパスワードで日本語のエラーが出る
- [ ] リロードしてもログイン状態が保持される
- [ ] その他タブ→ログアウト→ログイン画面に戻る

## 2. ペット(ホーム)

- [ ] 初回はたまご(ゆらゆら)が表示される
- [ ] ペットをタップ→ぷにっと潰れて吹き出しが変わる(なでる回数が減る)
- [ ] 「ごはん」→🍚が1減り、おなかゲージが回復する
- [ ] 🍚が0のとき→「ごはんがない」旨のトースト
- [ ] なでる残り回数が0になると「なでられすぎ」系のセリフになり回数が減らない
- [ ] Lv2到達でたまごが かえる(ベビーに進化+年表に記録)
- [ ] 20時以降(または設定で「いつも夜」)にすると部屋と全体が夜モードになる
- [ ] 0〜6時はペットが寝ていて zzz が出る

## 3. 2人同時の動作(重要)

- [ ] 片方がごはん/なでるをすると、**数秒以内にもう片方の画面にも反映**される
- [ ] 2人ともログインすると互いに「オンライン」表示(緑ドット)になる
- [ ] 片方がアプリを閉じると相手側で「○分まえ」表示に変わる
- [ ] 2人が同じ日に何か操作すると「💗のかけら+1」トーストが**1回だけ**出る
  (両方の端末で+2にならないこと)

## 4. 対戦(あそぶ)

- [ ] Aが「さそう」→Aは待機画面、Bにお誘いモーダルが出る
- [ ] Bが「またこんど」→両方とも元の画面に戻る
- [ ] Bが「あそぶ!」→両方に全画面のゲームが開く
- [ ] **いしんでんしん**: 同じ言葉(ひらがな/カタカナ/大文字小文字違い含む)で💞になる
- [ ] **心理コイン**: にぎる側/あてる側が毎ラウンド交代する
- [ ] 5ラウンド終了→結果画面→報酬(🪙・ペット✨)が付与される
- [ ] 結果画面を2回開き直しても**報酬が二重に増えない**
- [ ] 対戦中に片方が「中止」→両方とも部屋に戻る
- [ ] 対戦中に片方がアプリを閉じて90秒→相手側に「応答がありません」の帯が出る

## 5. はなす(質問デッキ)

- [ ] 「きょうの1問」が日付ごとに変わる
- [ ] 回答すると🍚+1、ペットに✨が入る
- [ ] 自分だけ回答→「あいて待ち」表示。相手の答えは**見えない**
- [ ] 2人とも回答→「みられる」になり、2人の答えが色分きで並ぶ
- [ ] 一度出した答えは変更できない

## 6. きろく

- [ ] きろく: ごはん・なでる・対戦・回答などが新しい順に色分きで並ぶ
- [ ] じっせき: 条件を満たすとトースト+紙吹雪で解除、グリッドが点灯する
- [ ] ごほうび: 条件達成で券が届く。「使う」→使用済みに移動する
- [ ] ねんぴょう: 誕生・進化・名前の変更などが時系列で並ぶ

## 7. その他

- [ ] 称号を装備するとプロフィールに表示される
- [ ] ペットの名前変更(10文字まで)が相手側にも反映される
- [ ] 記念日を設定→当日は「経験値2倍デー」バナーが出る
- [ ] きせかえ購入→残高が減り、装備するとペットに反映される
- [ ] 残高不足で購入→「足りない」トーストが出て減らない

## 8. 異常系

- [ ] 機内モードでボタンを押す→復帰後に整合が取れている(二重付与なし)
- [ ] 2人が**同時**にごはんを与える→ごはん残数・おなかが正しく2回分になる
  (トランザクションの巻き戻り確認)
- [ ] 第三者のアカウント(未登録メール)でFirebaseに直接アクセス→permission denied
